# Contenerización de Aplicación Web con Docker (WIP)

Este proyecto está **en desarrollo**.

## Objetivo
El objetivo es crear un entorno de desarrollo local reproducible para una aplicación web simple desarrollada en **Python (Flask)**.

El entorno se orquestará usando **Docker Compose** y constará de dos servicios:
1. Un contenedor para la aplicación Flask.
2. Un contenedor para una base de datos **PostgreSQL**.

## Tecnologías
- Python (Flask)
- Docker
- Docker Compose
- PostgreSQL
- Redis
